# Teslo API

### 1. Clone project
### 2. ```yarn install```
### 3. Clone ```.env.template``` file and rename it to ```.env```
### 4. Change environment variables
### 5. Run database start command
```
docker-compose up -d
```

### 6. Execute SEED 
```
http://localhost:3000/api/seed
```

### 7. Run: ```yarn start:dev```
